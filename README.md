# ZXBHELLO 网站

这是 ZXBHELLO 的个人网站和导航站，使用 [VuePress](https://vuepress.vuejs.org/) 和 [vuepress-theme-plume](https://github.com/pengzhanbo/vuepress-theme-plume) 构建生成。

## 功能特性

- 个人博客和文档站点
- 支持 GitHub 在线编辑
- 集成 Giscus 评论系统
- 支持 PDF 预览和嵌入资源
- 自动显示文档贡献者
- 响应式设计，支持移动端浏览
