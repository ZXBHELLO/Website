三角函数笔记

约 1178 字大约 4 分钟

数学三角函数学习笔记

2025-12-12

由于我总是容易忘记三角函数相关的公式，而这些公式通常形态相似、容易混淆，所以我决定整理一份详细的笔记，帮助自己加深理解并方便日后查阅。

一、三角函数的本质

三角函数的核心是 "单位圆上的坐标映射"，将角度（弧度制）与单位圆上点的坐标（x,y）绑定，本质是周期运动（如圆周运动、波动、机械振动）的数学描述，而非仅局限于直角三角形边角关系。

二、基础定义（双视角统一）

1. 单位圆视角（通用，覆盖任意角）

提示

单位圆是指圆心在原点，半径为1的圆。

设角θ的终边与单位圆交于点P(x,y)，则：

  • sinθ = y（纵坐标）
  • cosθ = x（横坐标）
  • tanθ = y/x（斜率）

2. 直角三角形视角（锐角限定）

  • sinθ = 对边/斜边
  • cosθ = 邻边/斜边
  • tanθ = 对边/邻边

与单位圆定义兼容：斜边=1时，x=邻边、y=对边

三、核心公式与规律（抓本质少记冗余）

1. 同角三角函数关系（3个核心，其余可推导）

  • sin²θ + cos²θ = 1（单位圆x²+y²=1）
  • tanθ = sinθ/cosθ（斜率定义）
  • 1 + tan²θ = sec²θ（推导得出，secθ=1/cosθ）

2. 诱导公式（核心规律："奇变偶不变，符号看象限"）

  • 奇偶性：

    • sin(-θ)=-sinθ（奇函数）
    • cos(-θ)=cosθ（偶函数）
    • tan(-θ)=-tanθ（奇函数）

  • 周期性：

    • sinθ、cosθ周期为2π
    • tanθ周期为π

    单位圆旋转1圈/半圈重复

3. 和差倍半公式（核心应用工具）

  • 和差公式：

    • sin(A±B)=sinAcosB±cosAsinB
    • cos(A±B)=cosAcosB∓sinAsinB

  • 二倍角公式：

    • sin2θ=2sinθcosθ
    • cos2θ=cos²θ-sin²θ=2cos²θ-1=1-2sin²θ（万能化简式）

  • 半角公式（由二倍角推导，可直接用）：

    • sin(θ/2)=±1−cosθ2\sqrt{\frac{1-cosθ}{2}}21−cosθ​​
    • cos(θ/2)=±1+cosθ2\sqrt{\frac{1+cosθ}{2}}21+cosθ​​
    • tan(θ/2)=sinθ1+cosθ\frac{sinθ}{1+cosθ}1+cosθsinθ​=1−cosθsinθ\frac{1-cosθ}{sinθ}sinθ1−cosθ​（无根号，更实用）

4. 三角恒等变换（核心转化工具）

（1）积化和差公式（将乘积转化为和差，便于化简）

  • sinAcosB = sin(A+B)+sin(A−B)2\frac{sin(A+B)+sin(A-B)}{2}2sin(A+B)+sin(A−B)​
  • cosAsinB = sin(A+B)−sin(A−B)2\frac{sin(A+B)-sin(A-B)}{2}2sin(A+B)−sin(A−B)​
  • cosAcosB = cos(A+B)+cos(A−B)2\frac{cos(A+B)+cos(A-B)}{2}2cos(A+B)+cos(A−B)​
  • sinAsinB = -cos(A+B)−cos(A−B)2\frac{cos(A+B)-cos(A-B)}{2}2cos(A+B)−cos(A−B)​

（2）和差化积公式（将和差转化为乘积，便于因式分解）

  • sinA + sinB = 2sinA+B2\frac{A+B}{2}2A+B​cosA−B2\frac{A-B}{2}2A−B​
  • sinA - sinB = 2cosA+B2\frac{A+B}{2}2A+B​sinA−B2\frac{A-B}{2}2A−B​
  • cosA + cosB = 2cosA+B2\frac{A+B}{2}2A+B​cosA−B2\frac{A-B}{2}2A−B​
  • cosA - cosB = -2sinA+B2\frac{A+B}{2}2A+B​sinA−B2\frac{A-B}{2}2A−B​

（3）万能公式（将任意三角函数转化为tan(θ/2)的代数式，统一变量）

设t = tan(θ/2)（θ≠π+2kπ，k∈Z），则：

  • sinθ = 2t1+t2\frac{2t}{1+t²}1+t22t​
  • cosθ = 1−t21+t2\frac{1-t²}{1+t²}1+t21−t2​
  • tanθ = 2t1−t2\frac{2t}{1-t²}1−t22t​

5. 升次降次公式（核心用于化简、积分，基于二倍角推导）

（1）降次公式（将二次方转化为一次方，简化运算）

  • sin²θ = 1−cos2θ2\frac{1 - cos2θ}{2}21−cos2θ​（由cos2θ=1-2sin²θ变形）
  • cos²θ = 1+cos2θ2\frac{1 + cos2θ}{2}21+cos2θ​（由cos2θ=2cos²θ-1变形）
  • sinθcosθ = sin2θ2\frac{sin2θ}{2}2sin2θ​（由sin2θ=2sinθcosθ变形）

（2）升次公式（将一次方转化为二次方，用于凑配平方）

  • 1 - cos2θ = 2sin²θ
  • 1 + cos2θ = 2cos²θ
  • sin2θ = 2sinθcosθ（也可看作"乘积升次"）

四、图像与性质（关键特征速记）

函数周期值域奇偶性核心单调区间（0-2π内）
sinθ2π[-1,1]奇函数增：(0,π2),(3π2,2π)(0,\frac{π}{2}),(\frac{3π}{2},2π)(0,2π​),(23π​,2π)；减：(π2,3π2)(\frac{π}{2},\frac{3π}{2})(2π​,23π​)
cosθ2π[-1,1]偶函数增：(π,2π)(π,2π)(π,2π)；减：(0,π)(0,π)(0,π)
tanθπR奇函数增：(0,π2),(π2,π)(0,\frac{π}{2}),(\frac{π}{2},π)(0,2π​),(2π​,π)（无减区间）

关键特征：

  • sinθ是"过原点的波浪线"
  • cosθ是"过(0,1)的平移波浪线"
  • tanθ是"渐近线分隔的递增曲线"（无最大值）

五、核心应用场景

  1. 几何领域：

    • 解三角形（正弦定理和余弦定理）
    • 向量点积（a⃗⋅b⃗=∣a⃗∣∣b⃗∣cosθ\vec{a}·\vec{b}=|\vec{a}||\vec{b}|cosθa⋅b=∣a∣∣b∣cosθ）
    • 图形旋转

  2. 物理领域：

    • 简谐运动（x=Asin(ωt+φ)）
    • 波动（声波/电磁波）
    • 圆周运动的线速度与角速度关系

  3. 代数化简：

    • 利用恒等变换、升次降次公式化简复杂三角函数表达式，求解方程

六、易错点警示

  1. 忽略定义域：

    • tanθ的定义域是θ≠π/2+kπ（k∈Z）
    • 万能公式中θ≠π+2kπ (无意义)

  2. 符号判断错误：

    • 半角公式、诱导公式需结合角所在象限确定正负
    • 参考"一全正，二正弦，三正切，四余弦"

  3. 弧度制与角度制混淆：

    • 所有公式默认弧度制，角度制需先转换为弧度

  4. 恒等变换误区：

    • 和差化积、积化和差公式易记混符号
    • 建议结合"和差与乘积的奇偶性"辅助记忆（如sinA+sinB结果含sin和cos，符号为正）
