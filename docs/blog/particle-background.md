## 概述

本站点已全局启用动态粒子背景效果。该效果为每个页面提供了一个生动的交互式粒子动画背景，增强了视觉体验。

## 功能特性

- ✨ **全局启用**：默认在所有页面显示动态粒子效果
- 🖱️ **鼠标交互**：粒子会对鼠标位置进行响应，创建连接线
- 📱 **响应式**：在移动设备上自动调整粒子数量以优化性能
- 🎯 **灵活配置**：支持逐页禁用粒子效果
- 🎨 **丰富的颜色**：粒子采用多种颜色，随机分布

## 使用说明

### 默认行为

页面默认显示粒子背景效果。访问任何页面时，都可以看到：
- 彩色的漂浮粒子
- 粒子之间的连接线
- 鼠标移动时粒子的响应效果

### 禁用粒子效果

如果需要在某个特定页面禁用粒子背景效果，只需在该页面的 frontmatter 中添加 `disableParticle: true` 配置：

```


```

### 示例

#### 启用粒子效果（默认）
```
---
title: 我的文章
date: 2024-01-01
---

这个页面会显示粒子背景效果。
```

#### 禁用粒子效果
```

title: 我的文章
date: 2024-01-01

这个页面不会显示粒子背景效果，用户可以更专注地阅读内容。
```

## 性能优化

粒子效果已针对不同设备进行了优化：

- **桌面设备**：最多显示 150 个粒子
- **移动设备**：根据屏幕宽度自动降低粒子数量（约 1.8 倍的因子）
- **渲染**：使用 `requestAnimationFrame` 实现平滑的 60fps 动画

## 技术实现

粒子背景效果是通过以下技术实现的：

- **HTML5 Canvas**：用于高性能的动画渲染
- **Vue 3**：用于组件管理和响应式数据处理
- **VuePress 2.0**：全局客户端配置

粒子效果支持以下交互：
- 自动移动和碰撞检测
- 粒子之间的连接线绘制
- 鼠标移动时的响应式连接
- 窗口调整大小时的自动重新计算

## 常见问题

### Q: 粒子效果会影响页面性能吗？

A: 粒子效果已进行了优化，使用 Canvas 进行渲染，对性能影响最小。在移动设备上会自动降低粒子数量以确保流畅体验。

### Q: 可以自定义粒子的颜色和数量吗？

A: 当前版本使用固定的配置。如需自定义，可以编辑 `docs/.vuepress/components/ParticleBackground.vue` 文件中的 `config` 和 `colors` 配置项。

### Q: 如何在某个页面临时禁用粒子效果？

A: 在页面的 frontmatter 中添加 `disableParticle: true` 即可。
