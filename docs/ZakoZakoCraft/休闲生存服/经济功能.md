## 经济系统功能总览

::: tip 指令参数说明
以下指令参数 \<> 表示必填项，[] 表示选填项。
:::

| 功能     | 指令                          | 说明                                                     |
| -------- | ----------------------------- | -------------------------------------------------------- |
| 查询余额 | `菜单` 或 `/balance [玩家名]` | 查询玩家的杂鱼币数量                                     |
| 转移金额 | `/pay <玩家名> <金额>`        | 向玩家支付                                               |
| 福布斯榜 | `菜单` 或 `/baltop`           | 杂鱼币排行榜                                             |
| 玩家交易 | `/trade`                      | 玩家间进行经验/杂鱼币/物品交易GUI                        |
| 系统商店 | `菜单` 或 `/shop`             | 官方商店，动态定价，价格随市场波动(通常高于玩家交易价格) |
| 头颅商店 | `菜单` 或 `/hdb`              | 出售装饰性头颅，用于建筑装饰                             |
| 物资回收 | `菜单` 或 `/sellgui`          | 一键出售物品给系统，价格参考系统商店                     |
| 红包系统 | `菜单` 或 `/hongbao`          | 将杂鱼币以红包的形式发送给玩家                           |
| 收集任务 | `菜单` 或 `/collections`      | 完成任务获取杂鱼币，内置排行榜                           |

## 注意事项

::: warning 重要提醒
- 禁止利用恶意复制漏洞
- 发现漏洞请及时反馈
- 交易请仔细确认物品与金额
- 纠纷请及时联系管理员
:::

::: info 经济系统理念
经济系统旨在促进玩家之间的互动与交易，营造活跃的游戏经济环境。
:::
