## 实用功能总览

::: tip 指令参数说明
以下指令参数 \<> 表示必填项，[] 表示选填项。
:::

| 功能     | 指令/获取方式           | 说明                                       |
| -------- | ----------------------- | ------------------------------------------ |
| 帮助指南 | `菜单`                  | 访问当前网页，含完整规则，新玩家必读       |
| 查询系统 | `菜单` 和 `/co i`       | 查看方块操作记录，解决纠纷专用             |
| 进阶系统 | `菜单`                  | 二十余种技能，涵盖生存、建筑、战斗..       |
| 安全保险 | `菜单`                  | 存放贵重物品的私人保险箱（堪比随身末影箱） |
| 随机传送 | `菜单` 和 `/rtp`        | 打开随机传送菜单，支持所有维度             |
| 地标系统 | `菜单` 和 `/wtp`        | 由玩家创建的传送点，高度可定制             |
| 玩家传送 | `菜单` 和 `/tpa`        | 玩家之间传送，支持所有维度                 |
| 鞘翅特效 | `菜单`                  | 帅气逼人的鞘翅粒子拖尾特效                 |
| 便携工具 | `菜单`                  | 含小创世神、便携工作台、镜像仿制           |
| 特殊配方 | `菜单`                  | 服务器独家合成配方，仅本服生效             |
| 生物运输 | `潜行+空手右键`         | 抱起生物并投掷                             |
| 全服点歌 | `菜单` 和 `/music help` | 购买飞行权限后可使用飞行功能               |
| 安心挂机 | `菜单`                  | 抗波动盔甲架，钻石剑无法破坏，避免损坏机器 |


## 注意事项

::: warning 安全提醒
- 飞行注意能量剩余，避免高空坠落
- 工具箱权限需使用杂鱼币租用
:::

::: info 功能设计理念
实用功能旨在提升玩家的游戏体验，让日常游玩更加便捷舒适。
:::
