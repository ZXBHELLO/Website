# ZakoZakoCraft建筑创造服
#### 专业建筑服务器，提供纯净创造环境，支持大型建筑项目

::: tip 服务器特色
- 提供超大的地皮创造环境
- 支持大型建筑项目开发
- 配备专业建筑工具和模组
- 拥有独立的世界和权限系统
:::

## 核心功能

::: info 功能概览
1. **地皮系统** - 提供独立建筑区域，支持多人协作
2. **Axiom公理** - 专业建筑辅助工具，提升建造效率
3. **创世神** - 大型地形编辑工具，支持复杂地貌设计
:::
