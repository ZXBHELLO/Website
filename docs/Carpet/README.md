## 什么是Carpet？

::: tip Carpet简介
Carpet Mod是一个开源的Minecraft模组，基于Fabric开发。它允许玩家从技术角度完全控制游戏中的各个方面，提供性能优化、游戏机制改进和调试监控等功能[^1]。

Carpet最初由gnembon开发，现在由社区维护。它通过添加各种规则和命令来扩展和增强游戏的可玩性[^2]。
:::
