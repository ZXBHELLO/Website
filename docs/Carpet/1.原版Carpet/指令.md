# Carpet指令

Carpet模组提供了丰富的指令系统，允许玩家和管理员控制游戏的各个方面。以下是对Carpet模组中主要指令的详细介绍。

## Carpet基础命令

### /carpet
Carpet模组的主要命令，用于查看和修改规则。

#### 用法
```
/carpet [规则名] [值]
```

#### 示例
```
/carpet movableBlockEntities true
/carpet hopperCounters true
```

### /script
脚本命令，用于运行或管理 Carpet 脚本。

#### 用法
```
/script [选项] [参数]
```

## 调试与监控命令

### /log
日志命令，用于实时监控游戏状态。

#### 功能
- 监控生物生成
- 监控实体行为
- 监控玩家操作
- 监控红石活动

#### 用法
```
/log [选项]
```

### /tick
游戏刻命令，用于控制游戏时间流逝速度。

#### 功能
- 加速游戏时间
- 暂停游戏
- 查看TPS信息

#### 用法
```
/tick [选项]
```

### /counter
计数器命令，用于统计和监控各种数值。

#### 功能
- 统计特定事件发生次数
- 监控资源收集情况
- 跟踪玩家活动数据

#### 用法
```
/counter [选项]
```

## 实用工具命令

### /player
假人命令，用于管理和控制假人玩家。

#### 功能
- 创建和移除假人
- 控制假人行为
- 设置假人权限

#### 用法
```
/player [假人名] [操作]
```

### /spawn
生成命令，用于控制实体生成。

#### 功能
- 生成特定实体
- 控制生成条件
- 设置生成参数

#### 用法
```
/spawn [实体类型] [数量] [位置]
```

### /profile
性能分析命令，用于分析游戏性能。

#### 功能
- 分析实体性能
- 分析区块性能
- 分析玩家性能

#### 用法
```
/profile [选项]
```
