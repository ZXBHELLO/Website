通过将成员添加到权限组中，您可以轻松地为多个成员分配相同的权限。这样，您就不必为每个成员单独设置权限，从而节省了时间和精力。

为了简化权限结构，当成员被添加到权限组后则**不可再单独为其设置权限**，如果希望单独设置组内某个成员的权限请将其移出权限组。

在领地的管理界面点击【权限组】，即可查看当前领地的所有权限组：
