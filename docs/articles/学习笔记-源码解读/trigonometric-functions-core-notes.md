> 由于我总是容易忘记三角函数相关的公式，而这些公式通常形态相似、容易混淆，所以我决定整理一份详细的笔记，帮助自己加深理解并方便日后查阅。

## 一、三角函数的本质

三角函数的核心是 "**单位圆上的坐标映射**"，将角度（弧度制）与单位圆上点的坐标（x,y）绑定，本质是周期运动（如圆周运动、波动、机械振动）的数学描述，而非仅局限于直角三角形边角关系。

## 二、基础定义（双视角统一）

### 1. 单位圆视角（通用，覆盖任意角）

::: tip 
单位圆是指圆心在原点，半径为1的圆。
:::


设角θ的终边与单位圆交于点P(x,y)，则：

- **sinθ = y**（纵坐标）
- **cosθ = x**（横坐标）
- **tanθ = y/x**（斜率）

### 2. 直角三角形视角（锐角限定）

- **sinθ = 对边/斜边**
- **cosθ = 邻边/斜边**
- **tanθ = 对边/邻边**

> 与单位圆定义兼容：斜边=1时，x=邻边、y=对边

## 三、核心公式与规律（抓本质少记冗余）

### 1. 同角三角函数关系（3个核心，其余可推导）

- **sin²θ + cos²θ = 1**（单位圆x²+y²=1）
- **tanθ = sinθ/cosθ**（斜率定义）
- **1 + tan²θ = sec²θ**（推导得出，secθ=1/cosθ）

### 2. 诱导公式（核心规律："奇变偶不变，符号看象限"）

- **奇偶性**：
  - sin(-θ)=-sinθ（奇函数）
  - cos(-θ)=cosθ（偶函数）
  - tan(-θ)=-tanθ（奇函数）
  
- **周期性**：
  - sinθ、cosθ周期为2π
  - tanθ周期为π
  
  > 单位圆旋转1圈/半圈重复

### 3. 和差倍半公式（核心应用工具）

- **和差公式**：
  - sin(A±B)=sinAcosB±cosAsinB
  - cos(A±B)=cosAcosB∓sinAsinB
  
- **二倍角公式**：
  - sin2θ=2sinθcosθ
  - cos2θ=cos²θ-sin²θ=2cos²θ-1=1-2sin²θ（万能化简式）

- **半角公式**（由二倍角推导，可直接用）：
  - sin(θ/2)=±$\sqrt{\frac{1-cosθ}{2}}$
  - cos(θ/2)=±$\sqrt{\frac{1+cosθ}{2}}$
  - tan(θ/2)=$\frac{sinθ}{1+cosθ}$=$\frac{1-cosθ}{sinθ}$（无根号，更实用）

### 4. 三角恒等变换（核心转化工具）

#### （1）积化和差公式（将乘积转化为和差，便于化简）

- sinAcosB = $\frac{sin(A+B)+sin(A-B)}{2}$
- cosAsinB = $\frac{sin(A+B)-sin(A-B)}{2}$
- cosAcosB = $\frac{cos(A+B)+cos(A-B)}{2}$
- sinAsinB = -$\frac{cos(A+B)-cos(A-B)}{2}$

#### （2）和差化积公式（将和差转化为乘积，便于因式分解）

- sinA + sinB = 2sin$\frac{A+B}{2}$cos$\frac{A-B}{2}$
- sinA - sinB = 2cos$\frac{A+B}{2}$sin$\frac{A-B}{2}$
- cosA + cosB = 2cos$\frac{A+B}{2}$cos$\frac{A-B}{2}$
- cosA - cosB = -2sin$\frac{A+B}{2}$sin$\frac{A-B}{2}$

#### （3）万能公式（将任意三角函数转化为tan(θ/2)的代数式，统一变量）

设t = tan(θ/2)（θ≠π+2kπ，k∈Z），则：

- sinθ = $\frac{2t}{1+t²}$
- cosθ = $\frac{1-t²}{1+t²}$
- tanθ = $\frac{2t}{1-t²}$

### 5. 升次降次公式（核心用于化简、积分，基于二倍角推导）

#### （1）降次公式（将二次方转化为一次方，简化运算）

- sin²θ = $\frac{1 - cos2θ}{2}$（由cos2θ=1-2sin²θ变形）
- cos²θ = $\frac{1 + cos2θ}{2}$（由cos2θ=2cos²θ-1变形）
- sinθcosθ = $\frac{sin2θ}{2}$（由sin2θ=2sinθcosθ变形）

#### （2）升次公式（将一次方转化为二次方，用于凑配平方）

- 1 - cos2θ = 2sin²θ
- 1 + cos2θ = 2cos²θ
- sin2θ = 2sinθcosθ（也可看作"乘积升次"）

## 四、图像与性质（关键特征速记）

| 函数 | 周期 | 值域   | 奇偶性 | 核心单调区间（0-2π内）                                                    |
| ---- | ---- | ------ | ------ | ------------------------------------------------------------------------- |
| sinθ | 2π   | [-1,1] | 奇函数 | 增：$(0,\frac{π}{2}),(\frac{3π}{2},2π)$；减：$(\frac{π}{2},\frac{3π}{2})$ |
| cosθ | 2π   | [-1,1] | 偶函数 | 增：$(π,2π)$；减：$(0,π)$                                                 |
| tanθ | π    | R      | 奇函数 | 增：$(0,\frac{π}{2}),(\frac{π}{2},π)$（无减区间）                         |

> **关键特征**：
> - sinθ是"过原点的波浪线"
> - cosθ是"过(0,1)的平移波浪线"
> - tanθ是"渐近线分隔的递增曲线"（无最大值）

## 五、核心应用场景

1. **几何领域**：
   - 解三角形（正弦定理和余弦定理）
   - 向量点积（$\vec{a}·\vec{b}=|\vec{a}||\vec{b}|cosθ$）
   - 图形旋转

1. **物理领域**：
   - 简谐运动（x=Asin(ωt+φ)）
   - 波动（声波/电磁波）
   - 圆周运动的线速度与角速度关系

1. **代数化简**：
   - 利用恒等变换、升次降次公式化简复杂三角函数表达式，求解方程

## 六、易错点警示

1. **忽略定义域**：
   - tanθ的定义域是θ≠π/2+kπ（k∈Z）
   - 万能公式中θ≠π+2kπ (无意义)

1. **符号判断错误**：
   - 半角公式、诱导公式需结合角所在象限确定正负
   - 参考"一全正，二正弦，三正切，四余弦"

1. **弧度制与角度制混淆**：
   - 所有公式默认弧度制，角度制需先转换为弧度

1. **恒等变换误区**：
   - 和差化积、积化和差公式易记混符号
   - 建议结合"和差与乘积的奇偶性"辅助记忆（如sinA+sinB结果含sin和cos，符号为正）
