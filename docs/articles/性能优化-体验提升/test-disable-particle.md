## 粒子效果禁用测试

这个页面已禁用粒子背景效果。你会注意到这个页面**没有**背景粒子动画。

### 对比说明

- **启用粒子效果的页面**（如首页、其他文章）：可以看到彩色粒子和连接线
- **禁用粒子效果的页面**（当前页面）：清晰的背景，便于专注阅读

### 适用场景

禁用粒子效果适合以下情况：
- 需要展示代码的页面
- 需要用户专注阅读的重要内容
- 包含大量表格或数据的页面
- 设备性能较低的用户访问

### 如何在你的页面中禁用粒子效果

在 markdown 文件的 frontmatter 中添加：

```
---
title: 你的页面标题
disableParticle: true
---
```

点击上方的"导航页"或其他文章链接回到显示粒子效果的页面进行对比！
