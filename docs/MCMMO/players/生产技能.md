# MCMMO生产类技能详解

::: tip
生产类技能包括修理、炼药、分解等，这些技能帮助玩家维护和制作装备。
:::

## Repair（修理）

修理技能允许玩家使用铁块修理损坏的工具和装备。

### 经验获取方式

使用MCMMO铁砧修理物品可获得修理经验。

### 使用方法

::: info
1. 放置一个铁块（默认的MCMMO铁砧）
2. 右键点击铁块打开修理界面
3. 选择要修理的物品进行修理
4. 每次修理消耗一个原材料（如修理铁剑消耗铁锭）
:::

### 特殊能力

#### Repair Mastery（修理精通）

::: info 被动能力
- **效果**：提升修理效果，修理量随技能等级提升而增加
:::

#### Super Repair（超级修理）

::: info 被动能力
- **效果**：修理时有一定几率触发超级修理，效果翻倍
:::

#### Arcane Forging（奥术锻造）

::: warning
- **解锁等级**：依赖于服务器配置
- **效果**：修理附魔物品时有一定几率保持附魔
- **机制**：
  - 成功保持附魔
  - 附魔等级降低（如效率V降为效率III）
  - 附魔完全消失
:::

## Alchemy（炼药学）

::: tip
炼药学技能增强酿造药水的能力。
:::

### 经验获取方式

使用酿造台酿造药水可获得炼药学经验。

### 特殊能力

#### Catalysis（催化）

- **效果**：大幅缩短药水酿造时间

#### Concoctions（调和）

- **效果**：解锁一些原本无法获得的特殊药水配方

## Smelting（熔炼）

熔炼技能增强熔炼矿物的能力。

### 经验获取方式

使用熔炉熔炼物品可获得熔炼经验。

### 特殊能力

#### Second Smelt（二次熔炼）

- **效果**：熔炼矿物时有一定几率获得双倍产物

#### Flux Mining（助熔挖矿）

- **效果**：挖矿时有一定几率直接获得熔炼后的产物（如直接获得铁锭而非铁矿石）

## Salvage（分解）

分解技能允许玩家将不需要的装备分解为原材料。

### 经验获取方式

使用分解功能分解物品可获得分解经验。

### 使用方法

1. 放置一个金块（默认的分解台）
2. 右键点击金块打开分解界面
3. 选择要分解的物品进行分解
4. 分解后获得制作该物品的原材料

### 特殊能力

#### Advanced Salvage（高级分解）

- **效果**：分解时有一定几率获得完整原材料，而非部分材料

#### Arcane Salvage（奥术分解）

- **效果**：分解附魔物品时有一定几率获得附魔书
