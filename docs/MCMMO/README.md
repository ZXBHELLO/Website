## 什么是MCMMO？

MCMMO是一个开源的Minecraft服务器插件，为游戏添加了RPG元素。玩家可以通过使用各种工具和技能来获得经验值并升级，解锁特殊能力和增强效果。

MCMMO的特点包括：
- 16种不同的技能系统
- 技能等级和经验值系统
- 特殊主动和被动能力
- 稀有战利品和掉落物
- 全新的游戏机制
- 更深入的PVP体验
