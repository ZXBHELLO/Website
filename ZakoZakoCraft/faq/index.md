::: warning 重要提醒
服主并非您的私人助理，没有义务逐一解答所有问题，请优先查阅本文档寻找答案。服主通常不会直接回应游戏内的 "服主为什么..."、"服主过来一下" 等简单询问。遇到问题时，请先尝试自行解决，或向其他玩家请教是否遇到过类似情况。只有在确实无法解决的情况下，才向服主寻求帮助。
:::
注册与登录
::: details Q：审核文档是必须填写的吗？
A：不是，加入机制包含审核制和邀请制两种方式。采用邀请制时，邀请者需对被邀请者的不当行为承担连带责任。
:::
::: details Q：服务器整合包中有什么内容？
A：包含生电核心模组套件（含 MASA 全家桶等经典工具）、元素周期表模组、性能优化模组，以及提升游戏体验的服务器联动模组（非必装，建议根据需求选择）。
:::
::: details Q：不同的线路有什么区别？
A：由于网络晚高峰可能导致延迟升高（属于正常网络现象），服务器提供多条网络线路供选择，可根据自身网络环境切换以优化连接质量。
:::
::: details Q：用于绑定的邮箱能使用其他邮箱吗？
A：不行，需强制使用数字QQ邮箱。此设置主要为便于追踪玩家行为（如查处违规操作） 及防范恶意机器人账号注册。
:::
::: details Q：绑定时不小心输入了错误的邮箱怎么办？
A：退出服务器并等待 10 分钟后重试即可，服务器会自动注销未注册成功的临时账号。
:::
::: details Q：忘记密码了怎么办？
A：通过已绑定的 QQ 邮箱，按照输入错误密码后系统提示的流程完成密码重置即可。
:::
::: details Q：我想更换绑定的邮箱该如何做？
A：使用指令/unregister <当前密码>注销账号后，重新绑定新的 QQ 邮箱即可（游戏内数据不会丢失）。
:::
::: details Q：我可以一个邮箱绑定多个 ID 吗？
A：可以，一个 QQ 邮箱最多可绑定 3 个游戏 ID。
:::
休闲生存服
::: details Q：生存服如何获取杂鱼币？
A：可通过以下方式获取杂鱼币：
在线奖励：每小时自动发放 20 杂鱼币 + 100 经验值（最便捷途径）
物资回收：在菜单中提交物资兑换杂鱼币（需持有对应资源）
收集挑战：完成菜单中的指定收集任务（奖励丰厚但耗时较长）
市场交易：在玩家市场出售稀缺资源（收益较高，需关注市场需求）
红包领取：接收其他玩家发放的红包（随机性较强，非稳定来源）
劳务服务：为其他玩家提供劳务获取报酬（需双方协商一致）
:::
::: details Q：服务器会开放假人（假玩家）功能吗？
A：目前暂不考虑开放假人系统。
:::
::: details Q：为什么跨区块区域会出现断崖？
A：这是服务器版本更新后区块生成机制变化导致的加载残留，属于正常现象，不影响游戏体验，无需特殊处理。
:::
::: details Q：生存服会删档吗？
A：不会进行全服删档，但会定期清理玩家加载时长不足 5 分钟的闲置区块（对正常游戏活动无影响）。
:::
::: details Q：服务器支持哪些类型的模组？
A：支持视觉优化、性能提升及操作便捷类模组。目前仅对投影打印机、轻松放置模组及菜单中列明的联动模组提供技术支持，其他类型模组不提供适配保障，相关问题将不予受理。
:::
::: details Q：我应该如何与其他玩家相处？
A：可根据个人偏好选择独居或群居模式，两种方式将带来不同的生存体验。无论选择哪种模式，都请务必与其他玩家和谐互动，避免产生冲突。
:::
::: details Q：如何创建和管理我的领地？
A：使用指令/dominion打开领地系统界面，通过图形化操作即可完成领地的创建与管理。详细规则请查阅领地系统文档。
:::
::: details Q：其他玩家可以破坏或拿走我领地内的物品吗？
A：不可以。领地建立后，仅领地所有者及授权玩家可执行破坏方块、拿取物品等操作。
:::
::: details Q：领地可以设置哪些权限？
A：可配置的权限包括但不限于：放置方块、破坏方块、使用门与按钮、攻击生物、操作容器等。所有权限均可在领地管理界面进行精细化设置。
:::
::: details Q：我可以拥有多个领地吗？
A：可以，但创建和维护多个领地需消耗相应资源。具体限制规则请参考领地系统文档。
:::
::: details Q：如何查看我的杂鱼币余额？
A：使用指令/balance可直接查询杂鱼币余额，也可通过游戏菜单中的个人中心查看。
:::
::: details Q：如何向其他玩家转账？
A：使用指令/pay <玩家名> <金额>即可向指定玩家转账。
:::
::: details Q：玩家市场如何使用？
A：使用指令/ah打开玩家市场界面，可在此出售自有物品或购买其他玩家上架的商品。每次最多可同时上架 20 件商品，交易成功后将收取 3% 手续费（最低 2 杂鱼币）。
:::
::: details Q：系统商店的商品价格会变化吗？
A：会变化。系统商店采用动态定价机制，价格将根据物品的市场供需关系实时调整（供不应求时价格上升，供过于求时价格下降）。
:::
::: details Q：我可以同时属于多个公会吗？
A：不可以，每个玩家仅能加入一个公会。
:::
::: details Q：我可以使用飞行功能吗？
A：可以。使用指令/flyc消耗经验值与杂鱼币兑换飞行时间即可开启。飞行时请留意能量剩余情况，避免因能量耗尽导致高空坠落。
:::
::: details Q：服务器支持哪些红石机制（生电机制）？
A：服务器支持大部分原版红石机制（即生存实用红石电路），包括 TNT 复制、重力方块复制、无头活塞等常见生电组件。详细支持清单请查看服务器公开配置项。
:::
::: details Q：如何获得更好的游戏性能？
A：建议优先使用服务器提供的官方整合包，其中包含针对性优化模组。同时可启用服务器联动功能进一步提升运行流畅度。
:::
::: details Q：如何获得更远的视距？
A：服务器默认渲染距离已优于同类型服务器，可保障基础游玩体验。若需更远视距，可安装Bobby 区块缓存模组实现扩展。
:::
