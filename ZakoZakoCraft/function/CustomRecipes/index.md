::: tip 特色配方介绍
服务器提供多种特色合成配方，让您的游戏体验更加丰富便捷。
:::

### 查阅方式：

* 在ZakoZakoCraft菜单中点击`特色配方`
* 输入命令 `/custom_recipe`

![图片](/assets/ZakoZakoCraft/CustomRecipes.png)
