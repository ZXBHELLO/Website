工会系统

约 1115 字大约 4 分钟

2025-10-07

提示

PlayerGuild 是一个专为 Minecraft 设计的公会插件，让玩家可以创建和加入公会，与朋友一起建设和发展公会。

简介

PlayerGuild 是一个简单易用的公会插件，玩家可以通过图形界面（GUI）轻松管理公会事务，无需记忆复杂的指令。

主要特性：

  • 可视化操作界面（GUI），简单直观
  • 公会成员管理
  • 公会升级系统
  • 公会战功能
  • 公会商城系统

基本操作

创建公会

要创建一个新公会，您可以使用以下任一方法：

  1. 直接在游戏中输入指令：/plg create <公会名称>
  2. 通过图形界面操作（推荐）

注意

创建公会需要消耗杂鱼币或满足特定条件，请确保您有足够的资源。

加入公会

加入公会的方式有几种：

  • 接受公会成员的邀请
  • 申请加入公开的公会
  • 使用指令：/plg accept <公会名称> 接受邀请

公会界面

大部分公会操作都可以通过图形界面完成：

  • 输入 /plg open 打开公会主界面
  • 在界面中可以查看公会信息、管理成员、升级公会等

主要功能

公会成员管理

  • 邀请新成员：/plg invite <玩家名>
  • 踢出成员：/plg kick <玩家名>
  • 成员权限管理

公会升级

随着公会发展，您可以使用 /plg up 指令升级公会，解锁更多功能和特权。

公会战

公会可以参与公会战：

  • 同意参战：/plg acceptPvp
  • 拒绝参战：/plg denyPvp

公会商城

公会可以拥有自己的商城系统，方便成员间交易特定物品。

常用指令

指令说明权限节点
/plg open打开公会主界面guild.use
/plg create <名称>创建新公会guild.create
/plg invite <玩家>邀请玩家加入公会guild.invite
/plg accept <公会>接受公会邀请guild.accept
/plg leave离开当前公会guild.leave
/plg up升级公会guild.upgrade
/plg list 或 /plg members查看公会成员guild.members

配置说明

服务器管理员可以通过修改配置文件来自定义插件行为：

  • 公会创建费用
  • 公会最大成员数
  • 升级所需资源
  • 战争规则设置

管理员专用指令

除了普通玩家可以使用的指令外，服务器管理员还可以使用以下高级指令：

指令说明
/plg reload重载配置文件
/plg create <公会名>创建一个公会
/plg open打开公会界面
/plg view <类型>管理员进行查看信息
/plg setLocation <类型> <场地名> <场地出生点名>设置公会战场地址
/plg addShop <价格>(等级)(次数)(类型)手持物品上架到公会商城
/plg addLogo <价格>(等级)手持物品上架到公会Logo商城
/plg restore <玩家名>用于蹦服等特殊情况获取玩家备份物品
/plg give <类型><名称><数量>增加贡献
/plg take <类型><名称><数量>减少贡献
/plg set <类型><名称><数量>设置贡献
/plg refresh刷新，用于修复公会没任务或者没称号
/plg dissolve <名称>强制解散指定名称的公会
/plg setEquipment (装备位置)设置手上物品为公平公会战物品
/plg start立即开始赛季公会战
`/plg convert <MySQLSQLite>`
/plg giveLogo <公会名>手持物品设置为对应公会logo
/plg setDefaultLogo手持物品设为公会默认logo
/plg channel切换到公会聊天频道
/plg adminCreate <公会名>OP创建公会
/plg adminEditGuildName <旧公会名><新公会名>OP修改公会名
/plg setRole <玩家名> <角色>OP修改玩家角色

常见问题

Q: 忘记公会名称怎么办？ A: 可以使用 /plg list 查看所有已加入的公会。

Q: 如何转让会长职位？ A: 会长可以在成员管理界面选择转让会长职位给其他成员。

Q: 公会可以解散吗？ A: 只有会长可以解散公会，使用 /plg disband 指令。

注意事项

温馨提示

  • 请友善对待公会成员，共同建设和谐的公会环境
  • 公会相关纠纷请联系服务器管理员处理
  • 离开公会前请妥善处理个人物品和贡献

小贴士

大部分操作都可以通过图形界面完成，即使不熟悉指令也能轻松使用公会系统！
